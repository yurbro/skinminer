# Fixture Bundles

Place manually curated evaluation fixtures here.

Recommended subdirectories:

- `text_only/`
- `table_only/`
- `figure_only/`
- `mixed/`

Each fixture bundle should ideally include:

- a stable fixture ID
- paper metadata
- the OA source file or a stable pointer to it
- notes describing why the paper belongs to this modality bucket
- any reviewer notes about ambiguity

Avoid storing run-generated files here unless they are intentionally frozen as benchmark artifacts.
